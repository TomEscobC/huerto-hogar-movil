# HuertoHogar - Frescura del Campo a tu Hogar

Este proyecto es una aplicación web para una tienda de productos frescos y orgánicos, construida con React y TypeScript.

## Estructura del Proyecto

- `frontend/`: Aplicación React totalmente funcional
- `assets/`: Recursos originales del proyecto estático (imágenes, CSS, etc.)

## Tecnologías Utilizadas

- React
- TypeScript
- Bootstrap
- React Bootstrap
- React Router DOM
- Bootstrap Icons
- @fontsource para fuentes Google

## Características

- Interfaz completamente responsiva
- Sistema de carrito de compras con localStorage
- Filtrado y búsqueda de productos
- Componentes reutilizables
- Navegación entre páginas
- Integración completa con Bootstrap para estilos

## Instalación

1. Navega al directorio `frontend`
2. Ejecuta `npm install` para instalar las dependencias
3. Ejecuta `npm start` para iniciar la aplicación en modo desarrollo
4. Visita `http://localhost:3000` para ver la aplicación

## Despliegue

Ejecuta `npm run build` para crear una versión optimizada para producción en el directorio `build`.
