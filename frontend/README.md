# HuertoHogar - React Application

Este es el frontend de la aplicación HuertoHogar convertido a React con TypeScript y Bootstrap.

## Características

- Aplicación React completamente funcional con TypeScript
- Bootstrap y React Bootstrap para componentes responsivos
- Bootstrap Icons para iconos
- Fuentes Montserrat y Playfair Display
- Sistema de carrito de compras con localStorage
- Navegación con React Router
- Componentes reutilizables

## Instalación

1. Asegúrate de tener Node.js instalado en tu sistema
2. Clona o descarga este repositorio
3. Navega al directorio `frontend`
4. Ejecuta `npm install` para instalar las dependencias
5. Ejecuta `npm start` para iniciar la aplicación en modo desarrollo
6. Visita `http://localhost:3000` para ver la aplicación

## Comandos Disponibles

- `npm start`: Inicia la aplicación en modo desarrollo
- `npm run build`: Crea una versión optimizada para producción
- `npm test`: Ejecuta las pruebas (si existen)
- `npm run eject`: Extrae las herramientas de configuración (no recomendado)

## Estructura del Proyecto

- `src/components`: Componentes reutilizables como Header, Footer, ProductCard
- `src/pages`: Páginas de la aplicación (Home, Products, Cart, etc.)
- `public/assets`: Recursos estáticos como imágenes y estilos

## Funcionalidades Implementadas

- Navegación principal con contador de carrito
- Página de inicio con productos destacados y filtros
- Página de productos con filtros y búsqueda
- Sistema completo de carrito de compras
- Páginas de autenticación (login/registro)
- Página de contacto
- Página de detalles del producto

## Tecnologías Utilizadas

- React (v18)
- TypeScript
- React Router DOM
- React Bootstrap
- Bootstrap Icons
- @fontsource para fuentes Google
